# Temel Operatörler

Java'da operatörler aşağıdaki gibi listelenebilir:

* **Atama Operatörleri**
* **Aritmetiksel Operatörler**
* **İlişkisel ve Eşitlik Operatörler**
* **Koşul Operatörler**
* **Mantıksal Operatörler**

## Aritmetik Operatörler

Matematiksel işlemleri programlama dilinde uygulamamızı sağlar.

* **Toplama:** a + b
* **Çıkarma:** a - b
* **Çarpma:** a * b
* **Bölme:** a / b
* **Mod Alma:** a % b
* **1 Arttırma:** a++
* **1 eksiltme:** a--

## Karşılaştırma Operatörleri

* **Eşitlik:** a == b
* **Eşit Değil:** a != b
* **Büyüktür:** a > b
* **Küçüktür:** a < b
* **Büyük Eşittir:** a >= b
* **Küçük Eşittir:** a <= b

## Mantıksal Operatörler

* **VE:** a && b
* **VEYA:** a || b
* **DEĞİL:** !(a&&b)

## Koşul Operatörü

* a = 5;
* b = (a == 1)? 1:0
* Çıktısı : 0
